# 영상공고 설치 안내

이 앱은 따로 프로그램을 내려받아 설치하지 않아도 됩니다. 아래 주소를 열고 브라우저에서 앱처럼 설치하면 됩니다.

앱 주소:

https://sdmidas-sudo.github.io/video-notice/

## 컴퓨터에 설치하기

1. Chrome 또는 Edge에서 앱 주소를 엽니다.
2. 주소창 오른쪽의 설치 아이콘을 누릅니다.
3. 설치 아이콘이 안 보이면 오른쪽 위 점 3개 메뉴를 누릅니다.
4. `앱`, `이 사이트를 앱으로 설치`, 또는 `영상공고 설치`를 선택합니다.
5. 설치가 끝나면 시작 메뉴나 바탕화면에서 `영상공고`를 실행합니다.

## 휴대폰 알림 받기

1. 휴대폰에 `ntfy` 앱을 설치합니다.
2. `+` 버튼을 누릅니다.
3. 서버는 `ntfy.sh`를 사용합니다.
4. 전달받은 주제 이름을 정확히 입력해 구독합니다.
5. 휴대폰 설정에서 ntfy 알림을 허용합니다.

## 새 공고 확인 방식

- GitHub Actions가 매일 오전 9시 17분쯤 자동으로 공고를 확인합니다.
- 새 공고가 있으면 ntfy로 휴대폰 알림이 갑니다.
- 공고 목록은 앱 주소에서 확인할 수 있습니다.

## 다른 사람이 직접 운영하고 싶을 때

다른 사람이 본인 GitHub 계정으로 직접 운영하려면 저장소를 Fork한 뒤 아래 설정을 해야 합니다.

1. GitHub Pages를 켭니다.
2. Actions 권한을 켭니다.
3. Repository secret `NTFY_TOPIC`에 본인 ntfy 주제 이름을 넣습니다.
4. Actions에서 `Daily notice scan`을 수동 실행해 테스트 알림을 확인합니다.
